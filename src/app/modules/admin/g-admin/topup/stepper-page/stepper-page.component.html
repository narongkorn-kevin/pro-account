วิธีการเติมเงิน

  • 1.คลิ้กลิ้งเพื่อรับ QR CODE
  • 2.เข้าแอพพลิเคชั่นของธนาคารในเครื่องมือสื่อสาร
  • 3.กดที่ปุ่ม "สแกน" หรือ "จ่ายบิล"
  • 4.สแกน QR CODE หรือ บันทึกรูปเพื่อสแกน
  • 5.ตรวจสอบข้อมูลการโอนเงิน
