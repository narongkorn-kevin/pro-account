รายการคำถามตอบ
เพิ่มคำถามตอบใหม่
จัดการ ลำดับ หัวข้อคำถาม คำตอบ สถานะ สร้างโดย วันที่สร้าง
1 1.Deeship คือบริษัทที่ทำด้านอะไร? เป็นบริษัทที่ให้บริการอย่างครบวงจรระหว่างตัวแทน และแม่ค้าออนไลน์. เปิดการใช้งาน Admin 2023/25/03 10:00:00
2 2.ลูกค้าจะมั่นใจในการรักษาความลับ และความปลอดภัยในการใช้งานได้อย่างไร? บริษัทได้นำระบบ Blockchain มาใช้ในการเก็บข้อมูลของลูกค้าอย่างรัดกุม และทั้งนี้ได้ร่วมมือกับบริษัทชั้นนำของโลกอย่าง เปิดการใช้งาน Admin 2023/25/03 10:00:00
3 3 มีสาขาอยู่ในต่างประเทศด้วยหรือเปล่า? มีแผนจะขยายธุรกิจไปในระดับโลก ทั้ง 190 ประเทศ เปิดการใช้งาน Admin 2023/25/03 10:00:00
4 4.การสะสมคะแนนในระบบต้องทำอย่างไร? เพียงซื้อครบทุกๆ 1000บาท ในบิลเดียวกัน รับ 1 คะแนน คะแนนจะพร้อมนำมาแลก Rewards เมื่อชำระเงินสำเร็จ. เปิดการใช้งาน Admin 2023/25/03 10:00:00
5 5.หากต้องการตรวจสอบตรวจสอบคะแนน ต้องทำอย่างไร? คะแนนสะสมของท่านจะแสดงอยู่แถบด้านบนของของแอพพลิเคชั่น หรือกดที่ปุ่มเมนูด้านซ้ายบนของแอพพลิเคชั่น ซึ่งคะแนนดังกล่าวจะเป็นคะแนนที่พร้อมแลก Deeship Rewards (ไม่รวมคะแนนจากการสั่งซื้อที่รอชำระเงิน). เปิดการใช้งาน Admin 2023/25/03 10:00:00
6 6.เหรียญสามารถนำไปใช้ประโยชน์อะไรได้บ้าง? สามารภซื้อขายแลกเปลี่ยนภายใน Ecosystem ของแพลตฟอรม์และเครือข่ายพันธมิตรได้. เปิดการใช้งาน Admin 2023/25/03 10:00:00
